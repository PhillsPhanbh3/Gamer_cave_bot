# Security Policy

## Supported Versions

Newest updates being version 2.0+

## Reporting a Vulnerability

To report a possible or confirmed security vulnerability within the Gamer Cave Discord bot, please open up a Security Vuln report here on github and the owner will get notification of it, reports will get acknowledged within 24 hours, and fixes within 1 week!

Please have a lot of detail in your security vulnerability report, with screenshots, what should happen, what is happening, and possible fixes.
